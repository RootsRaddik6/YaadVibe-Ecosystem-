Mobile: How to add files to the repo using GitHub Mobile App

1. Open GitHub app → open your repo (yaadvibe-ecosystem).
2. Tap "Add file" → "Create new file".
3. Enter file path exactly (e.g. src/app/layout.tsx).
4. Paste the file content from the chat into the edit box.
5. Commit (add a short message) → Commit to main branch.
6. Repeat for all files.
7. For images: Add file → Upload files → select from your gallery → commit.

After all files are added:
- Open Vercel → Import Project → select repository → set environment variables (use values from .env.example)
- Deploy
